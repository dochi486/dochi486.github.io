### 콘솔 타이틀 보더랜드 시리즈 모작

- 엑셀 데이터 연동하여 Scriptable Object 생성으로 기획 데이터 관리
- Post-Processing 적용
- 3D 공간에 UI 배치
- 몬스터 FSM 구현
- AWS EC 인스턴스에 MySQL 서버 구축하여 DB 활용
